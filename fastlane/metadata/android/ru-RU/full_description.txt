Calibre-Web-Companion
Неофициальное компаньон-приложение для Calibre Web и Calibre Web Automated, которое позволяет просматривать вашу коллекцию книг и управлять цифровой библиотекой в любом месте.

Создано с использованием <b>Flutter</b> и <b>Material You</b> для обеспечения современного, интуитивного и адаптивного пользовательского опыта.

<b>Поддерживаемые функции:</b>
<ul>
    <li>Просмотр библиотеки Calibre Web</li>
    <ul>
        <li>Просмотр книг по автору, серии, популярным и другим категориям</li>
        <li>Просмотр подробной информации о книге</li>
        <li>Доступ к подробной статистике коллекции</li>
    </ul>
    <li>Управление библиотекой</li>
    <ul>
        <li>Отмечать книги как прочитанные или непрочитанные</li>
        <li>Архивировать и разархивировать книги</li>
        <li>Организовывать книги на полках</li>
        <li>Управлять и настраивать ваши полки</li>
    </ul>
    <li>Чтение и устройства</li>
    <ul>
        <li>Загружать книги прямо на устройство</li>
        <li>Отправлять книги на электронную читалку (Kindle/Kobo) через send2ereader</li>
        <li>Использовать функцию почты Calibre Web для доставки на устройство</li>
    </ul>
    <li>Интеграция с автоматизацией</li>
    <ul>
        <li>Работает с Calibre Web Automated</li>
        <li>Автоматически загружать книги с помощью calibre-web-automated-book-downloader</li>
    </ul>
</ul>

Обратите внимание, что приложение находится в активной разработке. Вы можете сообщать об ошибках, давать обратную связь и предлагать идеи функций на GitHub.
